📖 사용법 · 규격 안내
· 프사 — SOOP 아이디만 넣으면 자동 연동. 직접 이미지 URL을 넣으면 그게 우선.
· 생일 — MM.DD (예: 06.08). D-Day가 자동 계산돼요.
· 방송 요일 — 체크한 요일만 방송으로 표시, 나머지는 자동 "휴방".
· 옷장 사진 — 세로 3:4 (900×1200 권장). SOOP 비공개 게시판에 올린 뒤 사진 우클릭 → "이미지 주소 복사"로 얻은 https://stimg.sooplive.com/… 주소를 넣으세요. (게시글 주소 ✗)
· 공지/일기 이미지 — 위와 같은 방법. 원본 글을 지우면 이미지도 깨집니다.
· VOD — SOOP 다시보기 주소의 숫자 ID만 입력 (예: vod.sooplive.co.kr/player/123456789).
· 저장 후 반영이 안 보이면 — 페이지에서 Ctrl+Shift+R (강력 새로고침).
